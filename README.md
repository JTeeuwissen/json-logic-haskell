JsonLogic library for Haskell

## Packages

Holds packages for:

* jsonlogic
* jsonlogic-aeson

## Run tests

Run all package tests:

```bash
cabal test all
```

## Build

build all packages:

```bash
cabal build all
```

## Documentation

To build all documentation run:

```bash
cabal haddock all
```
